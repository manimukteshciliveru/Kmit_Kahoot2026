# QuizMaster Pro - Documentation

## Project Overview
QuizMaster Pro is a high-performance, real-time quiz and competitive battle platform built with the MERN stack. It features AI-driven question generation, a 1v1 asynchronous "Racing Mode" Battle Arena, and a comprehensive ranking system for competitive learning.

---

## 🏗️ Architecture
- **Frontend**: React.js with Vite, Tailwind-style CSS, Lucide Icons, and Socket.io Client.
- **Backend**: Node.js, Express, MongoDB/Mongoose, Socket.io.
- **AI Engine**: Google Gemini (primary), Mistral AI (secondary fallback).
- **Authentication**: JWT-based secure sessions with role-based access control (Admin, Faculty, Student).

---

## ⚔️ 1v1 Battle Arena (Racing Mode)
The Battle Arena is a competitive feature where two players duel in real-time. Unlike traditional quizzes, it uses a decoupled **Racing Mode** flow.

### Key Logic:
- **Asynchronous Progression**: Each player answers questions at their own pace. As soon as an answer is submitted, the server provides the next question to that player immediately.
- **Winning Condition**: Decided after both players finish all questions or the timer expires. Calculated based on **accuracy (HP)** first, then **total score (Time Taken)**.
- **Damage System**: Correct answers deal damage to the opponent's HP; incorrect answers lead to self-damage.

### Ranking System:
- **ELO-Based Points**: Gain or lose points based on the opponent's skill level.
- **Rank Tiers**: Bronze (1-3), Silver (1-3), Gold (1-4), Platinum (1-4), Diamond (1-4), Heroic, and Grandmaster.
- **Win Streaks**: Bonus points and tiered progression are accelerated by maintaining win streaks.

## 🤖 AI Question Generation
The system utilizes a specialized AI service located in `server/services/battleAI.service.js`.

### Features:
- **Multi-Model Fallback**: Attempts `gemini-2.0-flash`, `gemini-1.5-flash`, `gemini-1.5-pro`, and `gemini-pro` sequentially. If Geminin is unavailable, it falls back to Mistral AI.
- **Contextual Generation**: Generates high-quality technical questions based on category (e.g., JavaScript, Python, DSA, DBMS) and level.
- **GeeksforGeeks/LeetCode Style**: Prompts guide the AI to simulate reputable source formats.

## 🔌 Socket.IO API Reference

### 1v1 Battle Events (`battle:`)
| Event Name | Type | Description |
|------------|------|-------------|
| `battle:searching` | Server -> Client | Notifies client they are in matchmaking. |
| `battle:started` | Server -> Client | Match found; provides questions, opponent info, and match metadata. |
| `battle:submit_answer` | Client -> Server | Sends answer index, question index, and time taken. |
| `battle:next_question` | Server -> Client | (Racing Mode) Sends the next question data immediately. |
| `battle:sync` | Server -> Client | Broadcasts HP and Score updates for both players. |
| `battle:ended` | Server -> Client | match result summary including point delta and streak updates. |

### Main Quiz Events (`quiz:`)
- `quiz:join`: Join a live session via Game PIN.
- `quiz:start`: Host signal to begin the quiz.
- `answer:submit`: Standard question submission.
- `quiz:sync`: Synchronizes state if a user reconnects.

## 📈 Ranking Tiers & Formula
- **Points to Level Up**: Standardized 100 points per tier jump up to Gold, then scales progressively higher for Platinum and Diamond.
- **ELO Calculation**: `pointsGained = (BaseFactor * (1 - ExpectedOutcome)) + StreakBonus`.
- **Top 500 Entry**: "Grandmaster" is reserved for the top rank-holders in the community.
